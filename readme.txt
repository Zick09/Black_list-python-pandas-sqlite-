-основной запускаемый файл main.py. в нем импортируются файлы с ddl и dml функциями
-запуск происходит с использованием аргумента. в качестве аргумента указывается дата файлов транзакций и черного списка (пример: 01052020)
-в результате выполнения, файлы копируются с приставкой "backup" и формируется отчет "report" с найдеными мошенниками
